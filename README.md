Instagram Clone Backend

This project is a backend application developed using Spring Boot. It replicates core functionalities of Instagram including authentication, posts, likes, comments, and follow system.

Features
- JWT-based authentication and authorization
- User registration and login system
- Create, delete, and manage posts
- Like & unlike functionality
- Comment system (add, update, delete)
- Follow / unfollow users

Technologies
- Java
- Spring Boot
- Spring Security
- JWT
- PostgreSQL

Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- DTO Structure

API Endpoints
- `/auth/register`
- `/auth/login`
- `/posts`
- `/comments`
- `/likes`
- `/follow`

How to Run
1. Clone the repository  
2. Configure `application.properties`  
3. Run the project  

Purpose
This project was built to demonstrate backend development skills using modern Java technologies and RESTful API design.
